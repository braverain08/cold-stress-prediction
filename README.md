# Cold Stress Prediction in Cold‑Region Open Fields using Multimodal Time‑Series Data and LSTM

This repository contains the source code, pre‑trained model, and dataset for the paper:  
Early Prediction of Cold‑Region Open Field Cold Stress Based on Multimodal Time‑Series Data and LSTM Network 
(Xiaoyu Xu, Huazhen Cui, Chaoyu Wang, Xu Zhang)

## Repository Structure

```
├── fill_and_prepare.py           # Data preprocessing & sliding window generation
├── train_lstm_complete.py        # Complete training script (fixed seed & gradient clipping)
├── test_pretrained.py            # Load pre‑trained model and reproduce paper results
├── lstm_model_complete.pth       # Pre‑trained model weights (reproduces paper numbers)
├── requirements.txt              # Python dependencies
├── README.md                     # This file
├── final_dataset.csv             # Original dataset (CC BY‑NC 4.0)
│
├── stability_analysis.py         # 30‑run stability analysis → Table 6, Figure 4
├── threshold_stability.py        # Threshold stability over 30 runs → Table 3
├── temporal_split_test.py        # Temporal subset evaluation → Table 7
└── plot_figure5_temporal.py      # Generate Figure 5 (temporal generalization bar chart)
```
## Requirements

- PyTorch 1.12 or higher

## Dataset Preparation

If you have the original CSV file `final_dataset.csv`, run the preprocessing script to generate the `.npy` files needed for training/testing:

```bash
python fill_and_prepare.py
```

This script will:
- Fill missing values (forward fill + mean imputation)
- Create sliding windows (past 7 days → predict cold stress 3 days later)
- Standardize features
- Save `X_train.npy`,  `y_train.npy`,  `X_test.npy`,  `y_test.npy`
> Note: If you only want to test the pre‑trained model, you still need to run this step once to obtain the test data files.

## Reproducing the Exact Paper Results

Due to the small training set (129 samples) and severe class imbalance (~10% cold stress), retraining the model from scratch may yield slightly different accuracy (68–83%) and precision (30–38%) across different runs. However, the 100% recall for cold stress is consistently achieved in all runs, and the relative performance hierarchy (LightGBM > LSTM > GRU) remains stable.

To exactly reproduce the numbers reported in the paper (accuracy 80.7%, precision 35.2%, recall 100%), we provide a pre‑trained model weight file `lstm_model_complete.pth` and a dedicated test script.

### Fast reproduction (no re‑training)

Ensure you have generated the test data files by running `fill_and_prepare.py` once.
Run the test script:
```bash
python test_pretrained.py
```
The script will output:

Test accuracy, recall, precision, macro F1 (same as Table 2)
Threshold table (single‑run, similar to Table 3)
Confusion matrix, PR curve, and threshold effect figures (same as in the paper)

## Reproducing All Stability and Generalization Tables/Figures

### Table 3 (threshold stability over 30 runs) & Figure 3 (updated threshold curve with error bands)

Run the threshold stability analysis:
```bash
python threshold_stability.py
```
This will:

- Train the LSTM 30 times with different random seeds
- Compute mean and std of recall/precision for thresholds 0.1–0.9
- Save results to `threshold_stability.csv`
- Print the table (matches Table 3 in the paper)
The updated Figure 3 (with shaded error bands) is generated by this script and saved as `threshold_stability.png`.


### Table 6 (stability statistics) & Figure 4 (boxplots)

Run the stability analysis:
```bash
python stability_analysis.py
```
This will:

- Train the LSTM 30 times
- Compute mean ± std, min, max for accuracy, recall, precision (cold), macro F1
- Print the statistics (matches Table 6 in the paper)
- Generate boxplots saved as `stability_boxplot.png` → Figure 4

### Table 7 (temporal subset performance) & Figure 5 (bar chart)
> Note: Before running `temporal_split_test.py`, ensure you have the pre‑trained model file `lstm_model_complete.pth` (generated by `test_pretrained.py` or `train_lstm_complete.py`).

Run the temporal split test:
```bash
python temporal_split_test.py
```
This will:

- Split the 2025 test set into May–Jun, Jul–Aug, Sep–Oct
- Evaluate the pre‑trained model on each subset
- Print accuracy, recall, precision, macro F1 (see Table 7)
- Save results to `temporal_performance.csv`
Then generate the bar chart:

```bash
python plot_figure5_temporal.py
```
This will produce `figure5_temporal_generalization.png` → Figure 5.
## Training from Scratch (Optional)
If you wish to retrain the model yourself, run:
```bash
python train_lstm_complete.py
```
The script will:

Fix all random seeds for reproducibility (`set_seed(42)`)
Train an LSTM for 150 epochs with class weights, dropout, gradient clipping, and ReduceLROnPlateau
Save the trained weights as `lstm_model_complete.pth`
Print the test metrics and threshold table
Due to the small dataset, you may observe moderate variability (accuracy 68–83%, precision 30–38%) across different runs. The 100% recall for cold stress is always maintained, and the conclusions about model comparisons and ablation studies remain unchanged.
